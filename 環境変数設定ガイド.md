# 環境変数の設定方法（簡易版）

## 📋 必要な3つの値

1. **Supabase URL** と **Supabase Key**
2. **OpenAI API Key**

## 🚀 クイックスタート

### ステップ1: Supabaseの設定値を取得

1. https://supabase.com にアクセスしてログイン
2. 「New Project」でプロジェクトを作成
3. プロジェクトが作成されたら、左メニューの「Settings」→「API」を開く
4. 以下の2つをコピー：
   - **Project URL**（例：`https://abcdefgh.supabase.co`）
   - **anon public** キー（長い文字列）

### ステップ2: OpenAI APIキーを取得

1. https://platform.openai.com/api-keys にアクセス
2. 「Create new secret key」をクリック
3. 表示されたキーをコピー（`sk-`で始まる文字列）
   - ⚠️ **重要**: この画面を閉じると二度と見れません！

### ステップ3: `.env.local`ファイルを編集

プロジェクトフォルダにある`.env.local`ファイルを開き、以下のように設定：

```env
NEXT_PUBLIC_SUPABASE_URL=https://あなたのsupabase-url.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=あなたのanon-key（長い文字列）
OPENAI_API_KEY=sk-あなたのopenai-key
```

### ステップ4: 設定の確認

ターミナルで以下を実行：

```bash
./scripts/check-env.sh
```

すべて✅が表示されればOKです！

## 📚 詳しい手順

詳細な手順は `ENV_SETUP_KITCHEN.md` を参照してください。

## ⚠️ 注意事項

- `.env.local`ファイルはGitにコミットしないでください（既に`.gitignore`に含まれています）
- APIキーは他人に共有しないでください
- OpenAI APIは使用量に応じて課金されます
