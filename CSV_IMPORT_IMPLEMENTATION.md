# CSVインポート機能 実装ガイド

## 実装概要

従業員の一括登録を可能にするCSVインポート機能を実装します。複数のフォーマットに対応し、他システムからの移行も容易にします。

---

## 対応フォーマット

### 1. らくポチ勤怠形式（自社形式）
**列の順序**：
```
社員番号, 氏名, 部署, 役職, メールアドレス, 電話番号, 生年月日, 住所, 入社日, 勤続年数, 有給付与日, 有給残数, 振込先口座, 交通費, 店舗, 勤務先住所, 権限
```

### 2. 一般的な形式（SmartHR風）
**列の順序**：
```
社員番号, 氏名, メールアドレス, 電話番号, 部署, 役職, 生年月日, 住所, 入社日, 権限
```

### 3. シンプル形式（最小限の必須項目）
**列の順序**：
```
社員番号, 氏名, メールアドレス, 電話番号, 住所
```

### 4. カスタムマッピング（将来的に拡張可能）
- ユーザーが列マッピングを設定できる機能

## 技術的な詳細

### CSVパース処理（手動実装例）

```typescript
function parseCSV(csvText: string): string[][] {
  const lines: string[] = []
  let currentLine = ''
  let inQuotes = false
  
  for (let i = 0; i < csvText.length; i++) {
    const char = csvText[i]
    const nextChar = csvText[i + 1]
    
    if (char === '"') {
      if (inQuotes && nextChar === '"') {
        currentLine += '"'
        i++ // 次の"をスキップ
      } else {
        inQuotes = !inQuotes
      }
    } else if (char === ',' && !inQuotes) {
      lines.push(currentLine)
      currentLine = ''
    } else if (char === '\n' && !inQuotes) {
      lines.push(currentLine)
      currentLine = ''
    } else {
      currentLine += char
    }
  }
  
  if (currentLine) {
    lines.push(currentLine)
  }
  
  return lines.map(line => line.split(',').map(cell => cell.trim().replace(/^"|"$/g, '')))
}
```

### フォーマット検出

```typescript
function detectFormat(headers: string[]): 'rakupochi' | 'general' | 'simple' | 'custom' {
  // ヘッダー行を分析してフォーマットを判定
  if (headers.includes('社員番号') && headers.includes('勤続年数')) {
    return 'rakupochi'
  }
  if (headers.includes('社員番号') && headers.includes('メールアドレス')) {
    return 'general'
  }
  if (headers.length <= 5) {
    return 'simple'
  }
  return 'custom'
}
```

## 必須項目のデフォルト値

CSVに含まれていない項目のデフォルト値：
- `role`: 'employee'
- `password`: 自動生成（後で変更を促す）または固定パスワード
- `isActive`: true
- `paidLeaveBalance`: 0

---

## パスワード生成方針

### オプション1: 自動生成
- ランダムパスワードを生成
- 初回ログイン時に変更を促す

### オプション2: CSVに含める
- CSVにパスワード列を追加
- ユーザーが設定したパスワードを使用

### オプション3: 固定パスワード
- 全員に同じ初期パスワードを設定
- 初回ログイン時に変更を促す

**推奨**: オプション1（自動生成）またはオプション3（固定パスワード）
