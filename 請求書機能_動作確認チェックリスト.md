### 4. 請求書作成ページ
- [ ] 請求先企業選択プルダウンが表示される
- [ ] 請求先企業を選択すると請求書番号が自動生成される
- [ ] 請求期間（開始日・終了日）が選択できる

- [ ] 対象従業員がチェックボックスで選択できる

- [ ] 「すべて選択」ボタンが動作する
- [ ] 件名、代金決済条件が入力できる
- [ ] 交通費、調整金額が入力できる
- [ ] バリデーションが動作する（必須項目チェック、日付の妥当性チェック）
- [ ] 請求書作成ボタンを押すと明細が自動計算される
- [ ] 請求書作成後、編集ページに遷移する

- [ ] 請求書の基本情報が正しく表示される
- [ ] 請求明細が正しく表示される
- [ ] 明細の各項目（勤務日数、基本単価、基本金額、残業時間、残業金額、欠勤日数、欠勤減算、遅刻・早退減算）が編集できる
- [ ] 明細を編集すると小計が自動再計算される
- [ ] 交通費、調整金額が編集できる
- [ ] 交通費・調整金額を変更すると合計金額が自動再計算される
- [ ] 消費税が正しく計算される（請求先企業の税率を使用）
- [ ] 保存ボタンが動作する
- [ ] ステータスが正しく表示される
- [ ] 下書き状態の場合は「発行する」ボタンが表示される
- [ ] 「発行する」ボタンでステータスが「発行済み」に変更される
- [ ] 発行時にPDFが自動ダウンロードされる
- [ ] PDFダウンロードボタンが動作する

### 6. 請求書PDF生成
- [ ] PDFが正しく生成される
- [ ] 請求書番号が正しく表示される
- [ ] 請求元企業情報が正しく表示される
- [ ] 請求先企業情報が正しく表示される
- [ ] 請求期間が正しく表示される

- [ ] 交通費が正しく表示される（該当する場合）

- [ ] 小計（税抜）が正しく表示される

- [ ] 合計（税込）が正しく表示される

### 7. 請求書コピー機能
- [ ] コピーボタンが動作する

- [ ] 件名に「（コピー）」が追加される
- [ ] 明細が正しくコピーされる
- [ ] コピー後、編集ページに遷移する

### 8. 勤怠データからの自動計算
- [ ] 請求期間の勤怠データが正しく取得される
- [ ] 勤務日数が正しく計算される
- [ ] 基本金額が正しく計算される（勤務日数 × 基本単価）
- [ ] 残業時間が正しく計算される（残業設定がONの場合）
- [ ] 残業金額が正しく計算される
- [ ] 欠勤が正しく判定される
- [ ] 欠勤減算額が正しく計算される
- [ ] 遅刻・早退が正しく判定される（15分以上）
- [ ] 遅刻・早退減算額が正しく計算される（1回あたり1000円）
- [ ] 明細の小計が正しく計算される

- [ ] 請求先企業未選択時にエラーメッセージが表示される

- [ ] 交通費が負の値の場合にエラーメッセージが表示される
