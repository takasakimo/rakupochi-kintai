# らくっぽ勤怠 — A4チラシ用（簡潔版）

> 中小企業の勤怠を、**正確に・楽に・法に沿って**。  
> クラウド型勤怠管理システム「らくっぽ勤怠」

---

## らくっぽ勤怠とは

従業員10〜50名規模の企業向けに特化した、Webベースの勤怠管理システムです。  
打刻の信頼性、労働基準法対応、シフトの「見える化」まで、必要な機能を一つにまとめています。

---

## 主な特徴（8つのポイント）

| 項目 | 内容 |
|------|------|
| **4段階打刻** | 起床→出発→出勤→退勤で、通勤・勤務時間を正確に記録 |
| **GPS位置情報** | 出勤・退勤時の位置取得で不正打刻を防止 |
| **タイムテーブル** | 日付ごとにシフトを時間軸で表示。ドラッグで編集・印刷可能 |
| **自動アラート** | 残業40h/60h超・連続勤務・有給失効前に通知 |
| **有給自動管理** | 会社ルールに合わせた付与・残数計算。シフト連動で自動更新 |
| **らくっぽリザーブ連携** | 勤怠で入力したシフトを予約管理側に自動反映（二重入力不要） |
| **マルチテナント** | 企業ごとにデータを完全分離。セキュアに運用 |
| **2〜24時間シフト** | 長時間勤務・夜勤にも対応した柔軟な勤務時間設定 |

---

## こんな企業に

- 従業員10〜50名程度の中小企業
- 複数店舗・事業所で勤怠をまとめて管理したい
- **シフトを一覧で確認・印刷して共有したい**（タイムテーブル）
- 労働基準法を守りながら、残業・有給をきちんと管理したい
- 初期費用を抑えて、クラウドで運用したい

---

## 他社との違い（差別化ポイント）

1. **4段階打刻** — 通勤・勤務を分けて記録し、労働時間を正確に把握
2. **GPS必須** — 出勤・退勤時の位置取得で打刻の信頼性を確保
3. **タイムテーブル** — シフトを時間軸で表示し、ドラッグ編集・印刷で現場運用を支援
4. **有給の自動管理** — 付与ルール・残数・シフト連動を自動化
5. **らくっぽリザーブ連携** — 予約管理と連携し、シフトの二重入力を解消
